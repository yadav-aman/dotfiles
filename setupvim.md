## Setup
```bash
cd ~
mkdir .vim
cd .vim
mkdir undodir
mkdir bundle
cd bundle
git clone http://github.com/VundleVim/Vundle.vim.git
cd ~
wget https://raw.githubusercontent.com/yadav-aman/dotfiles/main/.vimrc
vim ~/.vimrc
note - ignore errors and press enter
    once inside vim execute following command
:PluginInstall
```
